Today's Coding Session

Tired of seeing jumbled up code on your webpage? Let's make a code formatter for HTML, in Python!
  1. Import the system module and get your argument from command line.

    import sys

    filename = sys.argv[1]

  2. Replace every special character with their escape sequence

    with open(filename) as input_file:
     # Replace special characters with their escape character
     formatted_string = input_file.read().replace('\t', '&emsp').replace(' ', ' ').replace('<', '<').replace('>', '>').replace('\n', '<br>\n')
     # Write the result to another file
     with open(filename+'.txt', 'w+') as output_file:
      output_file.write(formatted_string)

  3. Now you have (less) jumbled up code to put in your webpage! Just run it in command line and give it a file name.
